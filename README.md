# 가족문화 역할극 경매 시스템 (Courtroom Auction)

## 🚀 실행 방법

이 프로그램은 실시간 통신을 위해 독립적인 백엔드와 프론트엔드를 사용합니다.
각각의 폴더에서 서버를 실행해주어야 합니다.

### 1. 백엔드(서버) 켜기
새 터미널(명령 프롬프트 또는 PowerShell)을 열고 아래 명령어를 입력합니다:
```bash
cd c:\Users\sohee\project\selling\auction-app\server
node index.js
```
`Socket.IO Server listening on port 3001` 메시지가 나타나면 정상입니다. (끄지 마세요)

### 2. 프론트엔드(웹) 켜기
또 다른 새 터미널을 열고 아래 명령어를 입력합니다:
```bash
cd c:\Users\sohee\project\selling\auction-app
npm run dev
```
`Local: http://localhost:5173/` 과 같은 주소가 나타나면 해당 링크를 인터넷 브라우저에 입력하여 접속합니다.

---

## 👨‍🏫 교사용 화면 (메인 화면)
- 모둠들의 활동을 통제하고 경매를 진행하는 화면입니다.
- 교실 앞 스크린에 띄워두고 진행하면 좋습니다.
- **[경매 시작]** 버튼을 통해 입찰을 받고, **[입찰 공개]** 버튼으로 학생들의 입찰액을 확인 및 낙찰하실 수 있습니다.

## 👥 학생용 화면 (모둠 화면)
- 각 모둠은 스마트폰이나 태블릿으로 위 주소에 접속하여 자신들의 모둠 번호를 선택해 입장합니다.
- 교사가 경매를 시작하면 비밀 입찰을 낼 수 있습니다.
- 3가지 항목(조건, 분위기, 장면)을 각각 하나씩 필수로 낙찰받아야 하므로 예산을 계산해서 입찰해야 합니다.
